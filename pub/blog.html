The Julia Blog

The Julia blog discusses issues of numerical, technical, distributed and parallel computing, as well as programming language design, and how these issues touch upon the design and implementation of the Julia programming language. Also see the Julia Computing for another source of information regarding ongoing Julia development.

Blogs from the broader Julia community can be found at Julia Bloggers.
